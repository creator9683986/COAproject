# Promo Service

## Описание
Promo Service отвечает за управление контентом платформы – промокодами и комментариями. Сервис реализует функционал создания, редактирования и удаления постов, а также поддержку вложенных комментариев (комментарии к комментариям).

## Зоны ответственности
- **Управление контентом:** Создание, редактирование, удаление и получение промокодов.
- **Работа с комментариями:** Добавление, обновление и удаление комментариев, включая возможность оставлять комментарии на комментарии.
- **Интеграция со статистикой:** Отправка событий (просмотры, лайки, комментарии) в Stats Service через брокер сообщений.
- **Валидация данных:** Проверка входных данных и обеспечение корректности операций над контентом.

## Границы сервиса
- Фокусируется исключительно на управлении постами и комментариями.
- Работает с собственной базой данных (PostgreSQL).
- Взаимодействует с gateway через gRPC для обработки запросов, связанных с контентом.

## Технологии и инструменты
- **Язык разработки:** C++
- **Контейнеризация:** Docker
- **База данных:** PostgreSQL
